# Sentiment_Analysis_System
Sentiment Analysis System

Objective:
To classify text into positive, negative, or neutral sentiment using NLP.

Technology Used:
- Python
- Natural Language Processing (NLP)
- Machine Learning (Naive Bayes)

Process:
1. Text preprocessing (lowercase, remove punctuation, stopwords)
2. Feature extraction using Bag of Words
3. Model training using Naive Bayes classifier
4. Sentiment prediction

Outcome:
The system accurately predicts sentiment from user-input text.

Conclusion:
This project demonstrates real-world application of NLP in opinion mining.
